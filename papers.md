## Общие данные и методы (что описано где)

### NOF эксперимент (calcium imaging, гиппокамп CA1)
- **Каноническое описание**: Гл. 2, секция 2.4.1 (part2.tex)
- **Животные**: мыши C57Bl/6, 3–5 мес., 4 дня по 10 мин в новой арене
- **Регистрация**: минископ UCLA Miniscope V4.4, GCaMP6s, GRIN-линза 1 мм
- **Число мышей**: 13 (eff_dim) или 16 (int_dim) — УТОЧНИТЬ, возможно разные выборки
- **Предобработка**: BEARMIND → CaImAn → NoRMCorre → CellReg → dF/F
- Упоминается в: eff_dim, int_dim, INTENSE
- **В других секциях** — только ссылка на каноническое описание, не повторять

### Процедура перемешивания (shuffle control)
- **Каноническое описание**: вместе с NOF
- Циклический сдвиг каждого сигнала на $\Delta > \Delta_{min}$ (5 с)
- Сохраняет структуру сигналов, разрушает межнейронные корреляции
- Используется в: eff_dim, int_dim

### фМРТ эксперимент (обусловленный страх)
- **Каноническое описание**: Гл. 2, секция 2.2 (part2.tex)
- 23 здоровых добровольца, 3T Siemens Verio, BNA atlas (245 ROI)
- RS_0 → обусловливание страха → угашение → RS_FE
- 300 объёмов × 245 ROI = 300 точек в 245-мерном пространстве
- Предобработка: FSL MELODIC, ICA-AROMA, FIX

## Статья 1: Эффективная размерность (eff_dim)

- **Ссылка**: Pospelov2024 (IEEE DCNA, конференция)
- **Исходник**: `parts/eff_dim/main_ru_passive.tex`
- **Глава**: 2, секция 2.4 (part2.tex)
- **Статус в диссертации**: текст перенесён в part2.tex ПОЧТИ ПОЛНОСТЬЮ

### Суть
Количественная оценка синхронизации нейронной популяции через эффективную размерность (ЭР) многомерного временного ряда кальциевой активности.

### Ключевые результаты
1. Разработан алгоритм расчёта ЭР с коррекцией спектра корреляционных матриц (итеративная коррекция по [spcorr])
2. ЭР растёт во время остановок животного (обратная корреляция со скоростью)
3. Перемешанная активность не показывает такой зависимости
4. Статистика: двухфакторная ANOVA (тип данных × день), оба фактора значимы
5. Первый день отличается от остальных (больше активности, короткие остановки)

### Ключевые формулы
- ЭР: $D_{eff} = (\sum_i \lambda_i)^2 / \sum_i \lambda_i^2$ (квадратичная энтропия Реньи, $q=2$)
- Коррекция спектра Марченко-Пастура для конечных данных

### Рисунки
- `images/effdim/eigs_act.jpg` — спектры собственных значений до/после коррекции
- `images/effdim/dim_speed.png` — ЭР vs. скорость животного (пример сессии)
- `images/effdim/stats.jpg` — статистика корреляций по дням

## Статья 2: Внутренняя размерность (int_dim)

- **Ссылка**: готовится к публикации (пока conference-стиль)
- **Исходник**: `parts/int_dim/int_dim_ru_passive.tex`
- **Глава**: 2, секция 2.4 (part2.tex), после eff_dim
- **Статус в диссертации**: текст перенесён в part2.tex ПОЧТИ ПОЛНОСТЬЮ

### Суть
Проверка гипотезы нейронного многообразия: активность гиппокампа лежит на искривлённом многообразии низкой размерности. Сравнение ЛР, ЭР и ВР; законы масштабирования ВР с числом нейронов.

### Ключевые результаты
1. ВР << ЭР << ЛР — подтверждение сильной нелинейности многообразия
2. ВР почти постоянна по времени (в отличие от линейных мер)
3. Степенное масштабирование $d \sim N^\alpha$ с малым $\alpha \in [0.1, 0.2]$
4. $\alpha$ для реальных данных значимо ниже, чем для перемешанных
5. Метафора «гимнастической ленты» для мультимасштабной размерности

### Методы
- ВР через оценщик 2-NN: $ВР = (n-1) / \sum \log(\mu_i)$ (MLE оценка)
- ЛР: PCA с порогом 95% объяснённой дисперсии
- ЭР: та же формула, что в eff_dim

### Рисунки
- `images/intdim/growth.jpg` — рост ЛР/ЭР/ВР с N и T
- `images/intdim/scaling.png` — степенное масштабирование ВР(N) по дням
- `images/intdim/slopes.jpg` — распределения α для реальных vs. перемешанных данных

### Связь с eff_dim
- Те же данные (NOF), та же предобработка, тот же shuffle
- ЭР здесь — one-shot (на всю сессию), а не скользящее окно
- В диссертации описание эксперимента НЕ дублировать

## Статья 3: фМРТ и снижение размерности (fMRI)

- **Ссылка**: Pospelov2021 (Neuroimage: Reports)
- **Исходник**: `parts/fmri/main_ru_passive.tex`
- **Глава**: 2, секция 2.2 (part2.tex)
- **Статус в диссертации**: СКЕЛЕТ, текст НЕ перенесён

### Суть
Нелинейные методы снижения размерности (Isomap, LLE, Laplacian Eigenmaps) для обнаружения изменений активности мозга в состоянии покоя после обусловливания страха.

### Ключевые результаты
1. PCA не различает RS_0 и RS_FE (IID ≈ 1.01)
2. Лапласовы собственные карты (LE) — лучший метод (IID ≈ 2.2 ± 0.5)
3. Вейвлет-удаление шума (Добеши-4, уровень 4) улучшает разделение в 2–3 раза
4. 19 из 23 участников показали разделение на индивидуальном уровне
5. Оценка внутренней размерности фМРТ данных: ID ≈ 6.1 ± 0.6

### Методы
- Manifold learning: PCA, Isomap, LLE, LE (лапласовы собственные карты)
- Вейвлет-сглаживание: DWT, Добеши, VisuShrink
- Метрика разделения: IID (межклассовое/внутриклассовое расстояние)
- Классификация: k-means, 10-fold CV

### Нарративная связка с Гл. 2
Переход от локальной популяции (calcium imaging) к масштабу всего мозга.
ROI играют роль отдельных нейронов, коллективная активность = синхронизация ROI.

## Статья 4: Максимизация активации в ИНС (neurotrain / MANGO)

- **Ссылка**: Pospelov2025 (Neurocomputing)
- **Исходник**: `parts/neurotrain/main.tex` (БОЛЬШОЙ файл, ~36k токенов)
- **Глава**: 3, секция 3.3 (part3.tex)
- **Статус в диссертации**: СКЕЛЕТ, текст НЕ перенесён
- **Глоссарий**: `parts/neurotrain/TERMINOLOGY_GLOSSARY.md`

### Суть
Безградиентная максимизация активации нейронов в импульсных (spiking) нейронных сетях через оптимизацию в латентном пространстве генеративной модели (GAN) с помощью TT-разложения. Первое применение безградиентной МА к ИНС.

### Ключевые результаты
1. PROTES (TT-оптимизатор) превосходит 15 альтернатив из Nevergrad
2. Фреймворк MANGO: генератор + оптимизатор + анализ MEI
3. Формирование селективных нейронов коррелирует с обучением сети
4. Динамика специализаций: от простых к сложным по мере обучения
5. Сравнение SNN vs. ANN: SNN формирует более специализированные нейроны
6. Потенциал применения к биологическим нейронам in vivo

### Ключевые компоненты
- **Генератор**: GAN (BigGAN для ImageNet, DCGAN для CIFAR-10)
- **Оптимизатор**: PROTES — TT-разложение для поиска в дискретизированном латентном пространстве
- **Анализатор**: косинусное расстояние между MEI, энтропия активации, сложность/сжимаемость
- **Фреймворк MANGO**: обёртка над всеми тремя компонентами

### Обучаемые сети
- Spiking ResNet-20 (LIF нейроны, суррогатный градиент) на CIFAR-10
- ANN ResNet-20 для сравнения
- 4 слоя анализа: conv1 + 3 блока (layer1.sn1, layer1.sn2, layer2.sn2)

### Рисунки
- `images/neurotrain/` — большая галерея (MEI, динамика, сравнения)
- Ключевые: `MANGO.png`, `emergence-of-specializations.png`, `fraction-of-selective-neurons.png`, `fig4_new.png`

## Статья 5: INTENSE (в работе)

- **Ссылка**: Pospelov2023 (Genes & Cells, тезисы) + полная статья в подготовке
- **Исходник**: НЕТ отдельного файла в `parts/` — будет писаться с нуля
- **Глава**: 3, секция 3.1 (part3.tex)
- **Статус в диссертации**: СКЕЛЕТ

### Суть
Информационно-теоретический метод для выявления нейронной селективности: взаимная информация (MI) между активностью нейрона и внешними переменными через копула-энтропию.

### Ключевые компоненты (из плана)
- Копула-энтропия для расчёта MI (непараметрический, не зависит от типа переменных)
- Статистическая значимость: синтетическое null-распределение MI через циклический сдвиг
- Поправки на множественные сравнения
- Анализ силы эффекта (в работе)
- Частичное информационное разложение (PID, в работе)

### Применения
- NOF данные: селективность к положению, скорости, объектам
- Селективность искусственных нейронов (RNN) — в работе
